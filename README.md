# Airbnb_Data_Analytics_Using_Python_PowerBi
🏠 Airbnb Data Analytics Project 
📌 Project Overview
This project provides a comprehensive end-to-end analysis of Airbnb listings. Using Python for data cleaning and exploratory analysis, and Power BI for interactive visualization, it aims to uncover key factors that influence rental prices and host performance.
# 🎯 Objectives
*  Clean and preprocess raw Airbnb data to ensure consistency.
*  Analyze pricing trends across different neighborhoods and property types.
*  Identify key features (like amenities or host status) that drive higher ratings and revenue.
*  Build an interactive dashboard to visualize findings for stakeholders.
  #  🛠️ Tech StackLanguage: 
  *  Python (Pandas, NumPy, Matplotlib, Seaborn).
  *  Visualization: Power BI Desktop.
# 📊 Analysis & Insights
* Pricing Factors:Discovered that room type (Entire home vs. Private room) is the strongest predictor of price.
* Seasonal Trends: Identified peak demand months where booking prices increase by over 20%.
* Geospatial Insights: High-density clusters of listings correlate with proximity to major landmarks.
